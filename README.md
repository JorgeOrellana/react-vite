# React App

Scaffolding for React Application with Typescript 

## Installation

Use the package manager [yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/) to install the dependencies.

```bash
yarn
```
or
```bash
npm install
```

## Scripts

### Dev Serve

```bash
yarn serve
```
or
```bash
npm yarn dev
```
